# Automated OMR Evaluation & Scoring System

A smart and efficient Optical Mark Recognition (OMR) system built using Python, OpenCV, and Flask to automate exam evaluation.

---

## Overview

At Innomatics Research Labs, placement readiness assessments are conducted for roles like Data Analytics and AI/ML for Data Science with Generative AI.

Manual OMR evaluation is time-consuming and error-prone. This project automates the entire process using image processing and web technologies, making evaluation faster, more accurate, and scalable.

---

## Features

- Automatic OMR sheet scanning  
- Bubble detection using OpenCV  
- Answer extraction and evaluation  
- Instant score calculation  
- Subject-wise performance analysis  
- Web interface for uploading OMR sheets  
- Clean and responsive UI using Bootstrap 5  

---

## Problem Statement

Each OMR sheet contains:

- 100 Questions  
- 5 Subjects  
- 20 Questions per subject  

Manual evaluation leads to inefficiency, delays, and human errors. This system automates detection, evaluation, and scoring to improve accuracy and efficiency.

---

## Tech Stack

- Python – Core backend logic  
- OpenCV – Image processing and bubble detection  
- NumPy – Numerical computations  
- Flask – Web application framework  
- Bootstrap 5 – Frontend styling  

---

---

## How It Works

1. Upload OMR sheet through the web interface  
2. Image is processed using OpenCV  
3. Bubbles are detected and analyzed  
4. Marked answers are extracted  
5. Answers are compared with the answer key  
6. Score and subject-wise analysis are generated  

---

## Output Example

- Total Score: 78 / 100  

Subject-wise Performance:

- Data Analytics: 16/20  
- AI/ML: 15/20  
- Aptitude: 17/20  
- Programming: 14/20  
- Reasoning: 16/20
